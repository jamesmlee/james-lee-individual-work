-this is the landing page

right click Web Pages --> New --> JSP
File Name = result
-this is the response page; displays results calculated by Servlet

expand Source Packages
right click on package; New --> Servlet
Class Name: "____Servlet"
-this acts as our controller
-uses data from index.jsp as input


=================================
Establish Component Communication
=================================
set up index.jsp
-edit title, h1
  <p>Please enter the number you want to factor:</p>
  <p>
  <form method="POST" action="FactorizorServlet">
      <input type="text" name="numberToFactor">
      <input type="submit" value="Find Factors!">
  </form>
  </p>

go to FactorizerServlet.java
-get rid of autogenerated code (i.e., response through try block)
-want to take submitted data and process it to make available to result.jsp
-only need to setAttribute for something you want to show in result.jsp
  String numberToFactor = request.getParameter("numberToFactor");
  request.setAttribute("numberToFactor", numberToFactor);
  RequestDispatcher rd = request.getRequestDispatcher("result.jsp");
  rd.forward(request, response);

go to result.jsp
-edit title, h1
-add stuff to display
-make a link back
  <p>            
      <a href="index.jsp">Factor Another One</a>
  </p>

go to FactorizerServlet.java
-declare variables needed
do calculations

=============
Result JSP
=============
add dependency for JSTL into pom file
  <dependency>
      <groupId>javax.servlet</groupId>
      <artifactId>jstl</artifactId>
      <version>1.2</version>
  </dependency>

add directive for taglib to top of result.jsp
  <%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
